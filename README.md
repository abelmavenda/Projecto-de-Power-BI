# Projectos-de-Power-BI
Estes projectos foram desenvolvidos no Power BI, uma ferramenta de BI da Microsoft que nos permite coletar, transformar, analisar e visualizar dados para apoiar na tomada de decisões estratégicas dentro das empresas/organizações. Nesta onda de ideias, desenvolvi dashboards (Painéis Visuais) que espelham a "Análise dos Dados" de empresas fictícias, na qual foram exportados dados de diferentes fontes (Data Science Academy, chatgpt e kaggle) realizando todo o processo de ETL no ambiente do Power Query.
